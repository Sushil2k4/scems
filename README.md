# SCEMS (Smart Campus Event Management System)

A web app for campus event coordination. Built with the Vite + React + Shadcn stack and deployed on Vercel.

**Live Link:** [https://scems-iota.vercel.app/](https://scems-iota.vercel.app/)

## 🛠 Tech
- **Frontend:** React, TypeScript, Tailwind CSS
- **UI:** Shadcn UI (Radix UI)
- **Build/Runtime:** Vite, Bun
- **Deployment:** Vercel

## 💻 Local Development

1. **Clone & Install:**
   ```bash
   git clone [https://github.com/Sushil2k4/scems.git](https://github.com/Sushil2k4/scems.git)
   cd scems
   bun install

2. **Run Dev Server:**
   ```bash
   bun dev

1. **Build:**
   ```bash
   bun run build
